HTML Tags

'div'
Division:The div tag defines a division or section in an HTML document. It is a container that allows you to group and style content.
'p'
Paragraph:This tag is used to add a paragraph of text.
'hr'
Horizontal:This is used to create a horizontal line in HTML Code.
'a'
Anchor:This tag creates a hyperlink.
'img'
Image:This tag is used to add an image to a HTML code.
